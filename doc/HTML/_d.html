Telnet-Client für IConnect T2

D Fehlermeldungen

Error #nnn

 
Während einer bestehenden Telnet-Verbindung ist der unbekannte Fehler nnn aufgetreten. Bitte teilen sie ASH mit, unter welchen Umständen sich diese Fehlermeldung reproduzieren läßt.
 
error creating socket: no memory

 
T2 konnte kein Socket anlegen. Vermutlich haben sie mit IConnect noch keine Verbindung zum Internet hergestellt.
 
host 'xyz' was not found

 
Der gewünschte Host konnte nicht erreicht werden, antwortet derzeit also nicht.
 
host 'xyz' was not resolved to an IP address

 
Der gewünschte Host ist unbekannt. Eventuell liegt nur ein Tippfehler vor - einfach noch einmal versuchen.
 
No host specified

 
Keiner der Kommandozeilenparameter wurde als Hostangabe erkannt.
 
No host specified (command line argument missing)

 
Es wurde keine Kommandozeile übergeben. T2 erwartet mindestens die Angabe des Hostnamens.
 
Not connected

 
Obwohl der Host erkannt wurde, konnte keine Verbindung zu ihm aufgebaut werden. Eventuell ist der Host derzeit überlastet - einfach später noch einmal versuchen.
 
Port for telnet service not found. ("services" in ETCPATH?)

 
IConnect konnte keine Portnummer für das Telnetprotokoll finden. Stellen Sie sicher, daß ETCPATH in IConf korrekt gesetzt ist, und daß im dort angegebenen Verzeichnis die mit PPP-Connect mitgelieferte Datei "services" liegt.
 
Sockets not installed

 
Sie haben vergessen, SOCKETS.PRG in ihrem AUTO-Ordner zu starten.
 
SOCKETS.PRG is too old

 
Das installierte SOCKETS.PRG ist leider zu alt. Bitte besorgen Sie sich ein Update bei ASH.
